# TFG - Clínica Nuvia

Aplicación web desarrollada en PHP para la gestión de servicios de una clínica estética y deportiva.

## Funcionalidades

- Login de administrador.
- Panel de administración.
- CRUD de servicios: crear, visualizar, editar y eliminar.
- Subida y gestión de imágenes.
- API en PHP para devolver servicios en formato JSON.
- Visualización pública de tratamientos.
- Filtrado por categorías: Estética, Deportiva, Infiltraciones y Bienestar.
- Páginas corporativas: inicio, nosotros, tratamientos, reservas y contacto.
- Enlaces legales: política de privacidad, cookies y aviso legal.
- Diseño responsive.
- Integración prevista con Treatwell para reservas online.

## Puesta en marcha

1. Clonar el repositorio.
2. Importar la base de datos MySQL.
3. Colocar el proyecto en la carpeta www de Laragon.
4. Iniciar Apache y MySQL desde Laragon o XAMPP
5. Acceder desde localhost.
