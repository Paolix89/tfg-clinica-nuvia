TFG - Clínica Nuvia

Aplicación web desarrollada en PHP para la gestión de servicios de una clínica estética y deportiva.

Funcionalidades

Login de administrador
Panel de administración
CRUD de servicios (crear, ver, editar, eliminar)
Subida y gestión de imágenes
Visualización pública de servicios
Filtrado por categorías (Estética y Deportiva)

Tecnologías

PHP
MySQL
HTML
Tailwind CSS

Autora

Paola - TFG Desarrollo de Aplicaciones Web
